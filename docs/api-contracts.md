# Campus Connect — Cross-Team API Contracts

> **Owner:** All Teams (collaborative document)
> **Purpose:** Document all API endpoints that are built by one team and consumed by another.
> **Rule:** Before building a cross-team endpoint, the owning team MUST document the contract here and get agreement from the consumer team.

---

## How to Add a Contract

Copy this template and fill it in:

```markdown
## [Module] — [Endpoint Name]
- **URL:** METHOD /api/v1/path
- **Owner:** [Faculty/Admin] Team
- **Consumer:** [Student/Faculty/Admin] Team
- **Auth required:** Yes — roles: [STUDENT, FACULTY, ...]
- **Query params:** param_name (type, required/optional)
- **Request body:** (if POST/PUT)
- **Response 200:**
- **Possible errors:**
- **Status:** Draft / Agreed / Implemented
```
